## When to use this skill

- When creating new database migration files
- When modifying existing migration scripts
- When adding, altering, or dropping database tables
- When adding, modifying, or removing columns from tables
- When creating or dropping database indexes
- When writing data migrations to transform existing data
- When implementing rollback/down methods for migrations
- When working with migration tools (Knex, Prisma, TypeORM, Drizzle, Sequelize, etc.)
- When editing files in migrations/, db/migrate/, or database/migrations/ directories
- When naming migration files with timestamps or sequential numbering
- When considering zero-downtime deployment strategies for schema changes
- When separating schema changes from data migrations
