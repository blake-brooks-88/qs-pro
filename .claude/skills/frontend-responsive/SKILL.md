## When to use this skill

- When building layouts that need to adapt to different screen sizes
- When implementing mobile-first CSS with progressive enhancement
- When using media queries or responsive breakpoint utilities
- When applying responsive Tailwind classes (sm:, md:, lg:, xl:)
- When creating fluid layouts with percentage-based widths
- When using relative units (rem, em) instead of fixed pixels
- When ensuring touch targets are appropriately sized (minimum 44x44px)
- When optimizing images and assets for mobile devices
- When testing UI across mobile, tablet, and desktop screen sizes
- When prioritizing content display for smaller screens
- When maintaining readable typography across all breakpoints
- When editing layout components, grid systems, or container elements
- When implementing responsive navigation menus or sidebars
- When using flexbox or CSS grid for responsive layouts
- When hiding or showing elements based on screen size
